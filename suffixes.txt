# grammatical voice
REFL                (I)n                  # reflexive voice (or pass.)
RECP                (I)ş                  # reciprocal voice
PASS                (I)l                  # passive voice
CAUS.1              t                     # causative type I
CAUS.2              DIr                   # causative type II
# REFL is used as PASS in some cases (when the verb ends in 'lV' with V avowel),
# CAUS.1 is used afer -l, -r, or a vowel in stems with multiplesyllables
# CAUS.2 is used elsewhere but there are many irregular forms which should be anaylzed as their ownroots
# The resultant meaning is constructive and not always apparent from the suffixes
# These suffixes can be chained: REFL+PASS, CAUS+CAUS=FAC (factitive), RECP+CAUS=REP (repetitive),
# CAUS.1+CAUS.2+CAUS.1 (causatives can be chained arbitrarily, alternatingly), REFL+PASS+CAUS, etc.
